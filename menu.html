Menus

A MooTools component to display responsive menus ( project repository ).

Simple menu

Here, each submenu has some directives contained in data-* attributes:

  • The data-popmode attribute contains "S=click&L=over", it means that the submenu must appear on click for S/M displays and on mouse over for L+ sized displays.
  • The data-popparent attribute contains "S=css&L=0", it means that the submenu positionning is done with CSS for S/M displays and relatively to the 0th parent element for L+ sized displays.
  • The data-poppos contains "L=leftbottomlefttop", it means that the submenu upperleft corner must be placed on the nth parent element's' bottomleft corner for L+ sized displays (the parent element has been defined above).
